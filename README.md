This project presents an interactive sales dashboard built using Power BI.
The goal was to transform raw sales data into clear business insights using visualizations and KPIs.

## Dashboard Features
Total Revenue Card
Total Boxes Shipped Card
Monthly Sales Trend (Line Chart)
Revenue by Country
Product Performance
Date Slicer for filtering

## Key Insights
Australia generated the highest revenue.
One product consistently dominated across months.
Sales declined between February and May before recovering in June.
Each country showed different product preferences.
